# 使用脚本

1.环境要求  
Python3.6或更高  
  
2.安装selenium  
在终端中使用pip install selenium  
  
3.编辑脚本内容  
设置文件保存位置（使用\\代替\）  
设置更改目标域名为实际目标（例如ambcn.us.kg）  
设置抓取范围（0-999）  
  
4.设置代理（可选）  
为防止IP被截获，可使用代理节点进行操作，请注意更新代理为全局  
  
5.在终端中启动脚本，等待脚本自动执行完成
